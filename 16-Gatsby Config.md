# React Gatsby

## Gatsby Config

### site-metadata

What casing on site-metadata when defining in gatsby-config.js

siteMetadata ( small 's' cap 'M' small 'd' )

```javascript
export default {
  siteMetadata: {
    title: `Slicks Slices`,
    siteUrl: 'https://gatsby.pizza',
    description: 'The best pizza place in Hamilton!',
  }
};
```
